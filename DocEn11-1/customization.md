# Customizations and extensions
      
In this chapter, you will learn about the options for modifying and extending the ALVAO Service Desk system. For information about customization options for common parts of Alvao, see [ALVAO Asset Management](../../alvao-asset-management/implementation/customization).

> [!TIP]
> To create software extensions, use the [ALVAO Service Desk Custom Apps](../../modules/alvao-sd-custom-apps) module.
